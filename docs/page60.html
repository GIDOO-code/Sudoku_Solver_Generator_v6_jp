Multi Secter Locked Sets(MSLS)の最初の考察


 Multi Secter Locked Sets(MSLS)

Multi Secter Locked Sets(MSLS)は、複数のセクター(行、列、ブロック)にまたがるロックを用いるアルゴリズムです。 MSLSは、 Locked Set , Fish などの Locked 系アルゴリズムの拡張です。

Fishではアルゴリズムの説明に BaseSet / CoverSet を用いますが、ここでは以下の MSLS に合わせて、Secter / Linkを用います。




MSLS は次のようなアルゴリズムです。

  • 盤面にいくつかの Secter を配置し、交差部分に着目します。

  • 交差部にある候補数字に着目し、それらを結ぶリンクをカウントします。
    候補数字を結ぶリンク表現が複数種あっても、1種類のみをカウントします。

  • 交差部の"セル数"と、交差部に入る候補数字をつなぐ"リンク数" が同数であれば交差部は Locked Set になります。
    次の例では 4セル⁼4リンクです。複数種のリンク表現 (c2 b1)#7 があります。

  • リンクと同じSecterの "交差部以外の候補数字" は、除外されます。

 Multi Secter Locked Sets(MSLS) の仕組み

  • 数独の Secter には、行・列・ブロック タイプがあります。セクターを構成するセルには、1～9の数字が重複なく配置されます。
  • 2つの secter-1, Secter-2 を考えます(次図)。secter-1, Secter-2 には交差する部分Cがあるとします。
  • 交差部Cにある候補数字(x)が肯定的に確定するなら、xは Aにはなく、Bにもない。
    これは、セクターには ”1～9の数字が重複なく配置される” ことから成立っします。



  • 複数Secterの集合が 2組ありこれらが交差している場合を考えます。
    交差部にある複数の候補数字が "Locked Set" である場合、 これらは 交差部内で肯定的に確定であり、交差部以外の同じセクターにある候補数字は除外されます。
    例えば、交差部内のセルのみで Fish が成立する場合、Locked Set になります。



  • 複数Secterの集合が交差しているとします。交差部Cで Locked が成立する候補数字 は、交差部Cのみで肯定的に確定します。

  • 交差部のFreeセル数 = リンク数 なら、交差部は Locked Set になります。
  • MSLSの Locked が成立します。また、Secter集合の交差部以外にある候補数字は除外されます。
