 Multi sector Locked Sets(MSLS)

Multi sector Locked Sets(MSLS)は、複数のセクター(行、列、ブロック)にまたがるロックを用いるアルゴリズムです。 MSLSは、 Locked Set , Fish などの Locked 系アルゴリズムの拡張です。

Fishではアルゴリズムの説明に BaseSet / CoverSet を用いますが、ここでは以下の MSLS に合わせて、sector / Linkを用います。




MSLS は次のようなアルゴリズムです。

  • 盤面にいくつかの sector を配置し、未確定数字の交差部に着目します。 sectorが3重に重なる交差部は対象外とします。

  • 交差部にある候補数字に着目し、それらを結ぶリンクを求めます。候補数字を結ぶリンク表現が複数種あることもあります。
    1つの候補数字のリンク群について、これらをカバーする最小のリンク群を求めます。このときの リンク数 をカウントします。

  • 交差部のセル数と、合計リンク数 が同数であれば交差部は Locked Set になります。 次の例では 5セル⁼5リンクです。

  • リンクと同じsectorの 交差部以外の候補数字 は、除外されます。

 Multi sector Locked Sets(MSLS) の証明

MSLS は次のように証明されます。

  • 前提:
    • 複数のセクターを含む集合を設定する(Aとする)。
    • 未確定交差部の候補数字を結ぶリンクを求める。(以下では、”未確定”を省略する）
    • 候補数字ごとに 最小のカバーリンクセットを求める。
    • 交差部のセル数と、カバーリンクセットのリンク数の合計 は同数とする。

  • 結論:
        If |cells of the intersection| = |links of the cover link set|, then A is Multi-sector Locked Sets (MSLS). ｜交差部のセル数｜=|カバーリンク数| なら A は Multi sector Locked Sets(MSLS)である。

  • 証明：
    • 交差部の セル数 と リンク数合計 が同数であることから、候補数字ごとの 最小リンク数 は確定する。
    • 従って、最小カバーリンクセットは、全リンクが肯定である。(各々のリンクには、肯定となる数字が必ず 1つある。)
    • Aは、Multi sector Locked Sets(MSLS) である。
全リンクは肯定であるが、交差部の候補数字が一意に決まるのではない ことに注意が必要である。 このことは、例えば、Fish と同じである。

 SK Loop

SK Loopは、8次のMSLSです。2つの行セクター、2つの列セクター、4つのブロックセクター構成です。
セクターの配置は、次の図のようになります。
SK Loopではセクターの3重の重なりがあります。ここは交差部外に扱い、候補数字があれば除外できます。
SK LoopはMSLSなので、解法アルゴリズムに特殊性はありません。sector集合を 2行2列4ブロック と指定することで、高速化が図れます。
